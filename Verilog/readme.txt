##공통 기능##
현재 임시로 기존의 하단 버튼(알람 클리어)를 스위치로 옮기고, 대신 동기화 버튼으로 사용중임
-----------------------------------------------------------------------------------------
버튼 3(상)을 누르면 모드 변경
[시계] -> [스톱워치] -> [타이머] -> [알람시계]

버튼 3(우) 길게 누르면 값이 빠르게 상승함
버튼 4(하단)을 누르면 buzzer를 끔
-----------------------------------------------------------------------------------------

##세부 기능##
-----------------------------------------------------------------------------------------
#시계#

'버튼 1'(중앙)을 누르면 모드 전환
[시계 모드] -> [세팅 모드]

[세팅 모드]에서는 시간이 흐르지 않고, 대신
'버튼 2'(좌)를 누르면 커서 전환(텍스트를 블링크하여 표시)
모드가 [시간 설정]일 때
초 -> 분 -> 시 -> 일 -> 월 -> 연
이 상태에서 '버튼 3'(우)를 누르면 커서 위치의 값이 1 증가
요일은 자동으로 계산됨

#스톱워치#
'버튼 1'(중앙)을 누르면 [정지], [시작] 전환
'버튼 2'(좌)를 누르면 lap
'버튼 3'(우)를 누르면 clear(00:00:00으로 초기화) 및 [정지] 모드로

#타이머#
'버튼 1'(중앙)을 누르면 설정한 값으로 [정지], [시작]전환 하여, [시작] 모드에서는 down counting함
00:00:00이 되면 buzzer가 작동(현재는 buzzer 대신 led로 출력중) 및 [정지]
단, 설정한 값이 없으면(00:00:00이면) 아무 동작도 하지 않음
'버튼 2'(좌)를 누르면 커서 전환
초->분->시
'버튼 3'(우)를 누르면 커서 위치의 값이 1 증가

#알람#
'버튼 1'(중앙)을 누르면 알람 [off], [on] 전환
알람으로 설정한 시간이 '시계' 모드의 시간과 같으면 buzzer가 작동함(현재는 buzzer 대신 led로 출력중)
'버튼 2'(좌)를 누르면 커서 전환
분 -> 시
'버튼 3'(우)를 누르면 커서 위치의 값이 1 증가
[on] 모드면 특정 아이콘 출력
-----------------------------------------------------------------------------------------
